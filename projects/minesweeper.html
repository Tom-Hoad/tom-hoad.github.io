Minesweeper

This is a recreation of the game Minesweeper. This project let me experiment with the flood fill algorithm, which is what is used to clear the grid when you click on an empty cell. Another addition to this game is a timer, which simply keeps track of how much time has passed on this current attempt.

The main menu features the option to change both the size of the grid and the number of bombs. These options are made in a way where you shouldn't be able to add an impossible number of bombs.

The graphics in this project were designed to recreate those of the Google Minesweeper game. This made use of a checkerboard which was created by taking the sum of the x and y coordinates and checking the modulo by 2. The borders around the grass also required particular checks of what cells were revealed.
