Yahtzee

This is a recreation of the classic game Yahtzee in JavaScript using the p5 library. It is an interesting project that was interesting to make. This is designed for 2 players, where you can click the roll button to roll the dice (the first roll is done for you), then each die can be clicked to move to the row below where it will be saved, meaning only the rest of the dice will be rerolled. Once all 3 rolls have been done (or before), you can click on a relevant row in the scoresheet to get the relevant number of points. While hovering the cell it will tell you how much you would get if you selected that cell.

The most interesting part of this project was creating the checks for each row. The upper section was fairly easy to check as you simply have to filter the dice for the amount of that number and then just add them. However, lots of the lower section was much more complicated and required more involved checks. This involved checking if there was a pair, 2 pairs and a 3 of a kind. Knowing these 3 things can solve a lot of the rows, but there are still some, like the small straight and large straight, which have more involved checks.
